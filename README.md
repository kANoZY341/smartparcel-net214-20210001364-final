# SmartParcel NET214 Final Project

SmartParcel is a serverless parcel delivery system built on AWS for NET214 Network Programming.

AWS Region: ap-southeast-2

## AWS Services Used
- API Gateway
- AWS Lambda
- DynamoDB
- S3
- SQS
- SNS
- CloudWatch
- CloudTrail
- CloudFormation
- VPC

## Features
- Create parcel
- Get parcel by ID
- List parcels
- Update parcel status
- Send email notification on status update
- Dead Letter Queue for failed messages
- Load test with 20 concurrent requests
